# 介绍

IFramework（任何C#环境均可）包含以下内容：

* Log （框架的log）
* Singleton （通用C#单例）
* PriorityQueue （优先级队列）
* Modules （模块合集）
* Environment （环境，包含模块合集与计时）


# 安装
* `git clone -b src https://github.com/OnClick9927/IFramework`
